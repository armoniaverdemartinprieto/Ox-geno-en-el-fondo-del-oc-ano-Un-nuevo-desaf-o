# Ox-geno-en-el-fondo-del-oc-ano-Un-nuevo-desaf-o
Un reciente estudio ha demostrado que el fondo del mar produce oxígeno. Este hallazgo plantea nuevos dilemas entre la comunidad científica y ambiental, ya que la minería submarina busca explotar los nódulos ricos en minerales esenciales, poniendo en riesgo el delicado equilibrio de los ecosistemas oceánicos.
